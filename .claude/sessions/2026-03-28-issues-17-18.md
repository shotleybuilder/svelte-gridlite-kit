### #20 follow-up — "invalid input syntax for type json"
- [x] Context menu "Filter by value" on JSONB cells used `equals` operator → `"skills" = $1` with JS object → invalid JSON syntax error. Fixed in `GridLite.svelte`: detect `json` dataType, use `jsonb_has_key`/`jsonb_not_has_key` instead, stringify objects.
- [x] `is_empty`/`is_not_empty` on JSONB columns generated `"col" = ''` which is invalid (empty string is not valid JSON). Fixed in `builder.ts`: added `::text` cast → `"col"::text = ''` — works for all column types including JSONB.
