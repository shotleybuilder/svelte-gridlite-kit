- `ColumnStateRow` interface: added `label: string | null`
- `saveColumnState()`: accepts optional `label` param, inserts into DB
- `loadColumnState()`: returns `label` field from stored rows

### `src/lib/GridLite.svelte`
- `customLabels` reactive state (Record<string, string>) loaded from DB on init
- `mergedColumnConfigs` derived: merges custom labels into columnConfigs for sub-components (FilterBar, SortBar, GroupBar, ColumnPicker, CellContextMenu)
- `getColumnLabel()`: checks customLabels first, then config, then column name
- Double-click on `<th>` label enters inline edit mode (`editingColumnLabel` / `editingLabelValue`)
- Commit on blur/Enter, cancel on Escape
- `persistColumnLabels()`: saves full column state including labels
- Row detail modal uses `getColumnLabel()` + `mergedColumnConfigs`

- `.gridlite-th-label-input`: styled inline input for label editing
